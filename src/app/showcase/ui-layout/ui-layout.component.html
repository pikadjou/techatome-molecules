Layout Page Structure Anatomy of a full page layout using ta-layout-* components 
ta-layout-page
├── ta-layout-header
│   ├── ta-layout-header-default [title] [showBack]
│   └── ta-layout-header-logo [src]
├── ta-layout-nav
├── ta-layout-content [autoHeight]
├── ta-layout-side
│   ├── ta-layout-side-content
│   └── ta-layout-side-cta [background]
└── ta-layout-panel          ← slot for ta-layout-with-panel

ta-layout-with-panel [open]
├── ta-layout-content         ← main area
└── ta-layout-panel           ← drawer content

ta-layout-full-panel [width] [title]  ← fixed side overlay
ta-modal [open] [size] [title]        ← centered modal overlay
