Layout Page Structure Anatomy of a full page layout using ta-layout-* components 
ta-layout-page
├── ta-layout-header
│   └── ta-layout-header-default [title] [showBack]
├── ta-layout-nav
├── ta-layout-content [autoHeight]
├── ta-layout-side
│   ├── ta-layout-side-content
│   └── ta-layout-side-cta [background]
└── ta-layout-panel
Layout Header Default ta-layout-header-default — Header bar with title and optional back button
With back button (showBack=true)
Without back button (showBack=false)
Layout Flex ta-layout-flex — Multi-panel flexible layout with left / center / right slots
Left Panel Navigation or sidebar content
Center Panel Main content area
Right Panel Details or actions panel
Layout Full Panel ta-layout-full-panel — Overlay side panel with title and close action Inputs: width (default 400px), title
Panel content goes here. This panel slides in from the side and provides a focused view.
Layout Title ta-layout-title — Section title component used within layouts
Page Section Title
Layout Page + Header + Content Composing ta-layout-page with ta-layout-header and ta-layout-content
This demonstrates a full page composition with header and content area.
Layout Side ta-layout-side with ta-layout-side-content and ta-layout-side-cta
Side Content This area holds navigation or filters.
