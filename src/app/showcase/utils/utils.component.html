appStopPropagation Prevents click events from propagating to parent elements
Parent (click me) @if (this.stopPropClicked) { Child clicked — parent NOT triggered } @else { Click the button to test }
appDnd Drag-and-drop file zone directive. Emits (fileDropped) event.
<div appDnd (fileDropped)="onFileDrop($event)">Drop files here</div>
ngLet Creates a local template variable from an expression (useful for async pipe results)
<div *ngLet="observable$ | async as data">{{ "{{ data }}" }}</div>
TaOnRender Conditionally renders content based on a boolean expression
<div [TaOnRender]="isVisible">Conditionally rendered content</div>
