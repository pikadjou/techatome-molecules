Overlay Panel Overlay panels attach floating content to a trigger element using the CDK overlay. They require runtime interaction and a panelConfig. Typical usage pattern: <ta-overlay-panel [panelConfig]="config"> <ng-template #panelTrigger> trigger element </ng-template> <ng-template #panelContent> floating content </ng-template> </ta-overlay-panel> The panelConfig accepts an OverlayMenuConfig with width, maxHeight, manualTrigger, and optional position overrides. The default panel component (ta-default-panel) renders the projected template inside a scrollable container.
