Usage Information Les boutons sont les éléments d'interaction principaux de l'interface. Utilisez les différents types en fonction du contexte :
  • Primary : Action principale de la page
  • Secondary : Actions secondaires
  • Action Button : Actions avec icône descriptive
  • Dual Button : Deux actions liées (ex: Save/Cancel)
