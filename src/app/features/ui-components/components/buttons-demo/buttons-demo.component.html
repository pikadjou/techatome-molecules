Utilisation Les boutons sont les éléments d'interaction principaux de l'interface. Utilisez les différents types en fonction du contexte et de l'importance de l'action.
  • Primary : Action principale de la page ou section
  • Secondary : Actions secondaires ou alternatives
  • Success : Actions de validation ou confirmation
  • Warning : Actions nécessitant attention
  • Danger : Actions destructives ou suppression
  • Outline : Actions tertiaires ou optionnelles
