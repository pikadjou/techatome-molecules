# `<ta-input-textarea>` — `TextareaComponent`

**Quand l'utiliser** : Rendu d'un `InputTextarea`.

**Usage direct** :
```html
<ta-input-textarea [input]="myTextareaInput" [standalone]="true"></ta-input-textarea>
```

**Notes** : Étend `TaAbstractInputComponent<InputTextarea>`. Utilise `InputLayoutComponent`. Aucun input supplémentaire.
