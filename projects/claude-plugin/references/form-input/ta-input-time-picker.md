# `<ta-input-time-picker>` — `TimePickerComponent`

**Quand l'utiliser** : Rendu d'un `InputTimePicker`.

**Usage direct** :
```html
<ta-input-time-picker [input]="myTimeInput" [standalone]="true"></ta-input-time-picker>
```

**Notes** : Étend `TaAbstractInputComponent<InputTimePicker>`. Utilise `NgxMaterialTimepickerModule`. Dépend de Angular Material Input.
