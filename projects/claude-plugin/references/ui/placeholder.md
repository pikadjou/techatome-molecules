# `<ta-placeholder>` — `PlaceholderComponent`
**Quand l'utiliser** : Skeleton loader (placeholder animé) pendant le chargement de contenu.
**Template canonique** :
```html
<ta-placeholder [placeholder]="skeletonConfig" />
```
**Notes** : Utilisé en interne par `<ta-loader>` via `skeleton` input. Rarement utilisé directement. La config `Placeholder` définit la grille de blocs animés.
