# `<ta-card-content>` — `CardContentComponent`
**Quand l'utiliser** : Zone de contenu principal d'une carte. Voir `_composed/card.md`.
**Template canonique** :
```html
<ta-card-content>
  <p>Contenu de la carte</p>
</ta-card-content>
```
**Notes** : Pas d'inputs. Fournit le padding et la zone de contenu via `<ng-content>`.
