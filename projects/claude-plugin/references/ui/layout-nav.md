# `<ta-layout-nav>` — `LayoutNavComponent`
**Quand l'utiliser** : Zone de navigation (sidebar/menu) dans `<ta-layout-page>`. Voir `_composed/layout-page.md`.
**Template canonique** :
```html
<ta-layout-nav>
  <ta-menu [routes]="routes" />
</ta-layout-nav>
```
**Notes** : Pas d'inputs. Conteneur pour le menu de navigation.
