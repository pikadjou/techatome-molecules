# `<ta-card-cta>` — `CardCtaComponent`
**Quand l'utiliser** : Zone d'actions (boutons) en bas d'une carte. Voir `_composed/card.md`.
**Template canonique** :
```html
<ta-card-cta>
  <ta-button (action)="onAction()">Voir plus</ta-button>
</ta-card-cta>
```
**Notes** : Pas d'inputs. Toujours en bas de la carte.
