# `<ta-tree-item>` — `TaTreeItemComponent`
**Quand l'utiliser** : Noeud d'arbre individuel. À utiliser dans `<ta-tree-container>`.
**Template canonique** :
```html
<ta-tree-item>
  Libellé du noeud
  <ta-tree-children>
    <!-- noeuds enfants -->
  </ta-tree-children>
</ta-tree-item>
```
**Notes** : Pas d'inputs. Utilise `<ng-content>`.
