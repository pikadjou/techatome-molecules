# `<ta-layout-not-found>` — `LayoutNotFoundComponent`
**Quand l'utiliser** : Page 404 standard avec bouton retour vers l'accueil.
**Template canonique** :
```html
<ta-layout-not-found />
```
**Notes** : Pas d'inputs. Extend `TaBaseComponent`. Clic "Retour" → `router.navigateByUrl('/')`.

**Note** : Sélecteur réel : `ta-layout-not-found` (pas `ta-not-found`).
