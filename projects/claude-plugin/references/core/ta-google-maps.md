# `<ta-google-maps>` — `MapComponent`
**Quand l'utiliser** : Carte Google Maps avec marqueurs clusterisés et calcul d'itinéraire.
**Template canonique** :
```html
<ta-google-maps></ta-google-maps>
```
**Notes** : Composant de démonstration avec marqueurs et itinéraire codés en dur. Requiert `provideGoogleMaps()` dans les providers de l'application. Utilise `@angular/google-maps` + `@googlemaps/markerclusterer`.
